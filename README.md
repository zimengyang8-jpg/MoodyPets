# 🐾 MoodyPets

MoodyPets is an iOS app that uses AI to analyze a pet’s mood from photos and provide human-readable insights. Users can upload images, receive mood predictions with explanations, and track past analyses through a personalized history view. The app combines AI inference with a clean SwiftUI interface and cloud-backed data storage.
### 🚀 Features
- Upload pet images from photo library
- AI-powered mood detection (e.g., Happy, Curious, Calm)
- Explanations + suggested actions for each result
- History view of all past analyses
- User authentication and personalized data
### 🛠️ Technologies Used
- SwiftUI – UI development
- Firebase Authentication – user login system
- Cloud Firestore – storing analysis history
- Firebase Storage – image storage
- OpenAI API (via Cloud Functions) – AI mood analysis
- Firebase Cloud Functions – backend processing
